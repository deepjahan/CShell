Compile and run the program "shell.c"
Features include:
1. Commandline with the directory from which code is run as home (~)
2. All the major builtin commands like cd, pwd, echo etc (no -e or -E flags for echo)
3. All the major system commands like ls, vim etc
4. Tried to implement foreground/background
5. Semicolon seperated commands are acceptable
6. On typing 'exit' or 'quit', the shell exits and program terminates
7. pinfo working with 0 or 1 arguments (ignore all the arguments after the first one)
